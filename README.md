# Social Media App

A full-stack social media application built with React, Vite.

## Features

- User authentication (login/register)
- Create, edit, and delete posts
- Like and comment on posts
- User profiles
- Responsive design

## Technologies

- Frontend:
  - React 18
  - Vite
  - Tailwind CSS
  - Axios
  - React Router
  - React Toastify
  - SweetAlert2

- Backend:
  - Json-server-auth
  - JSON Web Tokens (JWT)
  - Cloudinary (for image uploads)

## Setup

### Prerequisites
- Node.js (v16+)
- npm or yarn
- Cloudinary account (for image uploads)
